Takedown: Insane - TryHackMe Challenge Room

May 31, 2025 • TryHackMe • Insane

Mission Overview

A corporate webserver has been compromised by the RISOTTO GROUP. We've been authorized for cyber interdiction operations to locate their teamserver and take it down.

Static Analysis

Initial analysis of favicon.ico revealed interesting strings:

strings -n 6 favicon.ico | grep nim
fatal.nim
io.nim
parseutils.nim
strutils.nim
oserr.nim
os.nim

Key Finding:

The malware appears to be written in Nim, a relatively uncommon choice that might help evade detection.

Deep Dive Analysis

Further string analysis revealed C2 infrastructure details:

@[*] Sleeping: 10000
@results
@[*] Result: 
@http://takedown.thm.local/
@http://takedown.thm.local/api/agents/
@[*] Command to run: 
@[*] Hostname: 
@[*] My UID is: 
@[*] Key matches!

Environmental Analysis

  • Potential username discovered: c.oberst
  • API endpoints identified in C2 infrastructure
  • Evidence of environmental keying mechanisms
  • Automated command execution capabilities

Nim Code Analysis

The malware's structure follows typical Nim patterns:

  1. Entrypoint → main()
  2. main() → NimMain()
  3. NimMain() → NimMainModule()
  4. Additional WinMain() for Windows execution
