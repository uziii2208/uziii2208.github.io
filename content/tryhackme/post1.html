Room Overview

The Basic Pentesting room on TryHackMe is designed to teach fundamental web enumeration techniques. This write-up covers the essential tools and methodologies used in the initial phases of a web application security assessment.

Initial Enumeration

$ dirb http://TARGET_IP
$ nikto -h http://TARGET_IP
$ nmap -sC -sV -p- TARGET_IP

Web Directory Discovery

Using dirb, we discover several interesting directories:

+ /admin (Status: 301)
+ /dev (Status: 301)
+ /backup (Status: 301)
+ /secret (Status: 301)

Common Vulnerabilities

  • Directory listing enabled
  • Default credentials in admin panel
  • Sensitive files in backup directory
  • Misconfigured permissions

Exploitation Steps

  1. Enumerate web directories using dirb and gobuster
  2. Check for default credentials in admin panel
  3. Review source code for hidden information
  4. Download and analyze backup files

Tools Used

  • dirb - Web content scanner
  • nikto - Web server scanner
  • gobuster - Directory/file enumeration
  • nmap - Network exploration tool

Best Practices

  1. Always start with passive reconnaissance
  2. Document all findings systematically
  3. Verify findings manually to avoid false positives
  4. Follow proper authorization and scope guidelines
